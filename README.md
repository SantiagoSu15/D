# D
Doble rendija experimento


# Explicacion
El experimento consiste en hacer pasar luz  (en este caso un laser),a través de dos rendijas muy estrechas y observar el patrón que resulta en una pantalla detrás de las rendijas. Si la luz se comportara como partículas, se esperaría ver dos franjas correspondientes a las rendijas.Sin embargo, lo que se observa es un patrón de interferencia, lo que nos muestra  que la luz se comporta como una onda, con crestas y valles que se pueden sumar o cancelar entre sí. 
Cuando se realiza el experimento con partículas individuales, el patrón de interferencia aún se forma, lo que nos deja con varias dudas respecto a este fenomeno, una de las teorias mas aceptadas es que lo que afecta a la particula es una "historia" diferente de esta misma particula haciendo que cambie de camino, esto habre un supuesto paso a la posiblididad de la existencia de un multiverso.

Para el experimento se uso un laser, papel alumnio, cinta, y una caja
